Blink Application

This example introduces the simplest application that uses timer and LEDs. When started, the application makes all the MeshBean's LEDs blinking synchronously with a certain period.

The behavior of the application depends on the platform for which it is compiled. If development boards that fit platform configuration have buttons, like Meshbeans, then pressing the buttons increases or decreases the blinking frequency by a factor of two. If the boards do not have buttons the application is likely to behave in a different way: the frequency of blinking varies automatically within a certain range. The blinking frequency decreases from a default value by a factor of two every 10 seconds until the lower limit is reached and then increases by a factor of two with the same 10 seconds interval until the upper limit is reached. The limits as well as the initial value are given by special configuration parameters, which can be changed during compilation.

For detailed information please refer to BitCloud API Reference.
